Runtime contract

Tool permissions

Only enabled tools are exposed to the assistant and allowed to execute.

What this panel controls

You can turn tool families on or off here, but some access is still decided at runtime. Workspace tools stay inside the sandbox, canvas edit tools can pull in inspection helpers automatically, and web or news tools may be hidden when the user request does not call for them.

  • Turning a tool on allows the assistant to request it when context permits.
  • Turning a tool off removes it from the assistant's available options.
  • This page does not widen the workspace sandbox or override feature flags.
